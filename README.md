1. r/place 기능
    1. 명칭 정의하기
        - 스크린: 그림이 그려지는 공간, (n 칸 x m 칸) 크기를 가지고 있다. 기획에 따라 n개의 `스크린`이 이어질 수도 있음 
        - 픽셀: `스크린`을 구성하는 요소, 하나의 `색코드`를 가지고 있고 흰색이 default다.
        - 색코드: 색을 나타내는 코드, #FFFFFF 와 같이 6자리 16진수로 이루어진다. 위 프로젝트에서 이를 제한해서 몇 가지 색으로 진행해야 할 지 정해야 한다
    2. 기능 정의하기
        - 실시간 스크린 보기: `스크린`의 현재 상태를 볼 수 있다. 다른 사람이 `스크린`에 `색칠`하면 실시간으로 볼 수 있다.
        - 색칠: `스크린`의 한 `픽셀`을 색칠할 수 있다.
